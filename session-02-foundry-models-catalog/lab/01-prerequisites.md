# Prerequisites

- A Foundry project created in Session 1
- Access to the Foundry portal (ai.azure.com) with that project selected
- No additional quota, billing, or role assignments needed — browsing the catalog doesn't provision anything

## Before you start

Have a real task in mind (e.g., "summarize support tickets," "classify inbound emails," "generate SQL from natural language"). Browsing the catalog without a task in mind is how people end up shortlisting models on benchmark numbers alone. Pick something concrete from your own work if you can.
