# Implementation Steps

## Step 1: Sign in to the Foundry portal
Go to https://ai.azure.com and sign in with your Entra ID account.

## Step 2: Create a hub
1. Select **Create new** > **Hub**
2. Name it (e.g., `hub-agentic-ai-learning`)
3. Choose your subscription and region
4. Review and create

## Step 3: Create a project inside the hub
1. From the hub overview, select **Create new project**
2. Name it (e.g., `project-foundry-fundamentals`)
3. Confirm it's attached to the hub you just created
4. Review and create

## Step 4: Locate RBAC settings
1. From the hub, go to **Access control (IAM)**
2. Note the roles currently assigned
3. Navigate to the project and check its own **Access control (IAM)** blade
4. Compare — note which roles are inherited from the hub vs. assigned directly

## Step 5: Locate the Operate tab
1. From within the project, find the **Operate** tab in the left navigation
2. Open it and note the sections available (fleet overview, agents, models, tools)
3. Don't configure anything here yet — this is a preview for Sessions 7–8
