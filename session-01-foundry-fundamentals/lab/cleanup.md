# Cleanup

This session's resources are lightweight (no compute or model deployments), so cleanup is optional. If you want to remove them:

## Delete the project
From the project overview, go to **Settings** > **Delete project**.

## Delete the hub
From the hub overview, go to **Settings** > **Delete hub**. Note: you must delete all projects inside a hub before the hub itself can be deleted.

## Confirm deletion
Return to the Foundry portal landing page and confirm the hub no longer appears under "Recent hubs."
