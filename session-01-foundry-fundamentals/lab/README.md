# Lab: Foundry Fundamentals

## Lab overview

You'll create a Foundry hub and a project inside it through the portal, then locate (without configuring) the RBAC settings and the Operate tab.

## Files in this lab

- `architecture-flow.md` — how hubs, projects, and connections relate
- `prerequisites.md` — access needed
- `implementation-steps.md` — step-by-step portal walkthrough
- `validation-checks.md` — how to confirm each step worked
- `troubleshooting.md` — common access/permission issues
- `cleanup.md` — how to remove what you created
- `proof-of-execution.md` — template for recording your own run
