# Lab: Foundry Fundamentals

## Lab overview

You'll create a Foundry hub and a project inside it through the portal, then locate (without configuring) the RBAC settings and the Operate tab.

## Files in this lab, in order

Files are numbered to match the order you should open them. Work top to bottom.

1. `01-prerequisites.md` — access needed before starting
2. `02-architecture-flow.md` — how hubs, projects, and connections relate
3. `03-implementation-steps.md` — step-by-step portal walkthrough
4. `04-validation-checks.md` — how to confirm each step worked
5. `05-troubleshooting.md` — reference only, open if something breaks
6. `06-proof-of-execution.md` — fill in after you've completed the lab
7. `07-cleanup.md` — how to remove what you created, do this last

## Recommended flow

1. Read `01-prerequisites.md` and confirm access
2. Read `02-architecture-flow.md` to understand what you're about to build
3. Follow `03-implementation-steps.md` in order
4. Run through `04-validation-checks.md` after each major step
5. If something breaks, check `05-troubleshooting.md`
6. Fill in `06-proof-of-execution.md` with your own commands and output
7. Run `07-cleanup.md` before moving to the next session
